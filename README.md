# SOC Home Lab

## Overview
This repository documents my Security Operations Center (SOC) home lab setup used for hands-on cybersecurity practice, including threat detection, log analysis, and incident investigation.

## Purpose
To simulate real-world SOC environments for:
- Monitoring security events
- Investigating alerts
- Understanding attack patterns
- Practicing incident response

## Lab Setup Includes
- Kali Linux (attacker simulation)
- Windows/Linux systems (targets)
- Log generation and monitoring tools
- SIEM concepts (Splunk/Wazuh basics)

## Activities
- Simulated brute-force attacks
- Network scanning
- Log monitoring and analysis
- Incident investigation scenarios

## Skills Demonstrated
- SOC operations
- Threat detection
- Log analysis
- Incident response workflow
- Security monitoring

## Tools Used
- Kali Linux
- VirtualBox
- Wireshark
- Splunk / Wazuh (learning stage)
