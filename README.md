# SOC Home Lab Portfolio

## Overview
This repository documents my hands-on journey toward becoming a Security Operations Center (SOC) Analyst. It contains practical labs, investigations, documentation, and projects completed using industry tools such as Security Onion, Wazuh, Zeek, Sysmon, Wireshark, Splunk, and Python. Each lab reflects real-world SOC workflows, from alert triage and incident investigation to threat detection and reporting.

## Objectives

- Learn endpoint and network monitoring.
- Practice alert triage and incident response.
- Document investigations professionally.
- Develop a public cybersecurity portfolio on GitHub.
